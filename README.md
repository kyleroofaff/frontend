# Frontend (Local Dev)

## Run locally

- Install dependencies: `npm install`
- Start app: `npm run dev`
- Start frontend + backend together: `npm run dev:all`
- App URL: `http://127.0.0.1:5173`

## Launch and smoke testing

- Launch readiness checklist: `LAUNCH_CHECKLIST.md`
- Backend local test accounts and API startup: `../backend/README.md`
- Automated smoke checks (with frontend + backend running): `npm run smoke`
